1. Descripción del Servicio

TallerPro es una plataforma web de gestión para talleres mecánicos que permite administrar clientes, vehículos, reparaciones, inventario, presupuestos, mantenimientos programados y agenda de citas. El Servicio se ofrece como Software como Servicio (SaaS) accesible desde navegadores web y dispositivos móviles.

  • Cada taller puede tener un administrador que gestiona empleados y clientes.
  • Debés tener al menos 18 años para crear una cuenta.
  • Nos reservamos el derecho de suspender o cancelar cuentas que violen estos Términos.

3. Planes y Pagos

  • TallerPro ofrece un período de prueba gratuito de 14 días con acceso a todas las funciones Premium.
  • Al finalizar el período de prueba, debés elegir un plan de pago para continuar usando las funciones avanzadas.
  • Los precios están publicados en la sección "Plan" de la aplicación y pueden cambiar con aviso previo de 30 días.
  • El pago se realiza de forma mensual mediante transferencia bancaria o billetera electrónica.
